นายสิทธิกร ชิรงค์ (กร)

สาขาวิศวกรรมคอมพิวเตอร์และปัญญาประดิษฐ์

กลุ่มเรียน: วันพฤหัสบดี

เหตุผลที่อยากเรียนวิชา Full stack web development :

1.อยากเพิ่ม เรียนรู้และทำความเข้าใจทักษะใหม่ๆให้กับตนเอง
2.อยากลองทำ Front-end และ Back-end เพราะตอนดูเว็บคนอื่นบางทีก็คิดว่าเราอยากสร้างแบบเขาได้บ้าง เราต้องทำอะไรและทำยังไงบ้าง
3.เป็นหนึ่งในการตัดสินใจการเลือกอาชีพในอนาคต

ความคาดหวังต่อวิชานี้ :

1.เข้าใจกระบวนการทำงานทั้ง Front-end และ Back-end
2.รู้หลักการใช้ Syntax การทำ Database ต่างๆที่จำเป็นสำหรับ Full stack web development
3.ได้ทำโปรเจ็คที่สามารถนำไปใส่ใน Resume เพื่อใช้พิจารณาเข้าบริษัทได้จริง

อะไรจุดอ่อน ที่ตนเองต้องพัฒนาเกี่ยวกับทักษะ Programming และ แนวทางการพัฒนา :

1.พื้นฐานเบื้องต้นยังไม่แน่น แนวทางการพัฒนาคือศึกษาพื้นฐานที่จำเป็นต่อการใช้งาน ทดลองทำและทำความเข้าใจ
2.จำ Syntax ไม่ค่อยได้ แนวทางการพัฒนาคือฝึกใช้ Syntax ให้ครอบคลุมกับเนื้อหาที่ต้องใช้และใช้ให้หลากหลาย
3.ยังรู้ภาษาในการเขียนที่น้อย แนวทางการพัฒนาคือศึกษาภาษาโปรแกรมอื่นๆเพิ่มเติมจากทั้งในห้องเรียนและนอกห้องเรียน
